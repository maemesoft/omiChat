# omiChat-client
omiChat 채팅 클라이언트

## Prerequisites
 - Node.js (Tested LTS v12.18.4)
 - react-native-cli
 - expo-cli
 
## Development Stack
 - Typescript
 - Prettier (not using Linter)
 - expo bare workflow (SDK v39.0.2)
 - UI-Kitten V5
 - React-Navigation V5
 - Redux (with Redux-persist)
 - GraphQL (Apollo Client)
 - localforage (faster then localstorage in web)
